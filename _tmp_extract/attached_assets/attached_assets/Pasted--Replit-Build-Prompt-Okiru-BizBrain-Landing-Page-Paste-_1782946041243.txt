**Primary target keywords (optimise headings and body copy around these — these are realistic to rank for):**

Do not try to rank the H1 or title tag on the broad secondary terms alone — anchor them on the primary long-tail list above, where competition is realistic.

**Required technical SEO implementation:**
- Unique `<title>` tag under 60 characters using a primary keyword: e.g. "Okiru [PRODUCT_NAME] — All-In-One Business Tool for South African Small Businesses"
- Meta description under 155 characters, primary keyword included, matching the existing okiru.biz meta pattern (see toolkit.html for house style)
- Single H1 matching the hero headline, containing a primary keyword naturally

- Schema.org JSON-LD markup:

  - `FAQPage` schema wrapping the FAQ section
  - `BreadcrumbList` schema if the existing site uses breadcrumb navigation
- Add the new page to the existing sitemap.xml
- Confirm the page is NOT blocked in robots.txt
- Internal linking: add a contextual link to this new page from within the AI Tool Advisor flow — specifically, when a user's stated goal relates to managing a small business, running leads, invoicing, or social media, surface this page as one recommended option alongside genuine third-party tools (do not replace neutral recommendations, add to them)

- Ensure mobile responsiveness matches existing site breakpoints — test at 375px, 768px, 1024px
- Optimise for Core Web Vitals: compress any images used, lazy-load below-the-fold images, no layout-shift-inducing elements, avoid render-blocking scripts in the head
- Add descriptive, human-readable URL slug (avoid IDs or query strings)
